## Restore the environment

### Arch Linux / macOS

```console
$ git clone git@github.com:SlashNephy/.github.git ~/.github
$ bash ~/.github/script/install.sh
```

### Windows

```console
$ git clone git@github.com:SlashNephy/.github.git ~\.github
$ powershell ~\.github\script\install.ps1
```
