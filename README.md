# .github

My dotfiles and GitHub Action workflows collection

## Get Started

### Arch Linux / macOS

Requirements:
  - git
  - zsh

```console
$ git clone git@github.com:SlashNephy/.github.git ~/.github
$ zsh ~/.github/env/install.zsh
```

### Windows

Requirements:
  - git
  - powershell

```console
$ git clone git@github.com:SlashNephy/.github.git ~\.github
$ powershell ~\.github\env\install-windows.ps1
```
