# .github

My dotfiles and GitHub Action workflows collection

## Get Started

### Arch Linux / macOS

Requirements:
  - git
  - zsh

```console
$ git clone https://github.com/SlashNephy/.github ~/.github
$ zsh ~/.github/env/install.zsh
```

### Windows

Requirements:
  - git
  - powershell

```console
$ git clone https://github.com/SlashNephy/.github ~\.github
$ powershell ~\.github\env\install-windows.ps1
```
